Next3 Filesystem Home Page

This is the home page for the Next3(TM) filesystem. Next3 is a clone of the Ext3 filesystem with built-in support for snapshots. Next3 was developed by CTERA Networks

The Next3 Sourceforge summary page can be found here.

Documentation and other resources can be found here.

Release 1.0.12 of Next3 is available!

Featuring a new Makefile that builds a stand-alone kernel module, installs pre-compiled (x86) next3 utils and runs next3 snapshot tests on a loop mounted image.

Successfuly tested on Ubuntu desktop releases 9.10 and 10.4 and with stable kernel versions 2.6.27 to 2.6.33.
