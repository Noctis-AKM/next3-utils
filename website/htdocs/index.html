Next3 Filesystem Home Page

This is the home page for the Next3™ filesystem. Next3 is a clone of the Ext3 filesystem with built-in support for snapshots. Next3 was developed by CTERA Networks

The Next3 Sourceforge summary page can be found here.

Documentation and other resources can be found here.
